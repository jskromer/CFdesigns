# Counterfactual Design for M&V

Three decisions define every M&V plan: **Boundary**, **Model Form**, and **Duration**. This interactive course teaches you to make those choices with professional judgment — not just protocol compliance.

Based on *The Role of the Measurement and Verification Professional* by Steve Kromer (River Publishers, 2024).
